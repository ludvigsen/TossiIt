# Play Store assets (Organizel)

This folder contains **source SVGs** and **generated PNGs** for Google Play Console uploads:

- `organizel-icon-512.png` (512×512) — "High‑res app icon"
- `organizel-feature-graphic-1024x500.png` (1024×500) — "Feature graphic / promo graphic"

## Regenerate

From repo root:

```bash
cd web/marketing
./render.sh
```
